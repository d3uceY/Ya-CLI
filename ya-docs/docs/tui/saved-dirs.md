# Saved Directories

Saved directories let you store frequently-used paths and reference them by name. They are shared with [Ya-GUI](https://github.com/d3uceY/Ya-GUI) via `config.json`.

---

## Opening saved directories

Press `D` (capital D) from the shortcuts list.

---

## Saved directories keys

| Key | Action |
|-----|--------|
| `↑` / `k` | Move up |
| `↓` / `j` | Move down |
| `a` | Add a new saved directory |
| `d` / `x` | Remove selected directory |
| `Esc` / `D` | Go back to the shortcuts list |

---

## Adding a directory

Press `a` to open the add form:

- **name** — a short label (e.g. `projects`, `api`)
- **path** — the full path (e.g. `C:\Users\you\Projects` or `~/code/myapp`)

Press `Tab` to move between fields, `Enter` to save.
